## Модули системы
### 1. product.py
**Класс Product** - представление товаров:
- `name`, `price`, `quantity`
- Метод `__str__()` для отображения информации

### 2. warehouse.py  
**Класс Warehouse** - управление складом:
- `add_product()` - добавление товара
- `remove_product()` - удаление товара
- `show_products()` - просмотр товаров

### 3. store.py
**Класс Store** - управление магазином:
- `summa` - баланс магазина
- `add_product()` - добавление товара
- `sell_product()` - продажа товара
- `show_products()` - просмотр товаров
- `show_summa()` - отображение баланса
